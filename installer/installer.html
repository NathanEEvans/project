Grab an Installer     GUI | CLI

We have 2 installer types, one with and one without GUI. The CLI version will be best used for server installation but can also run fine on any machine running whatever-shell. The GUI version, well ... you know.

While aiming at creating one installer for each platform (Operating System), since we use software which runs on (virtualy) any platform, we still need to start somewhere.

So the first one to kick off with is written for the platform i'm working with the most these days: Ubuntu.

The installer will run for many distro's as is, not just Ubuntu, so if you used it and can verify that it worked out well on another distro you can add it in the Tracker so i can add it to the list as verified. Add the OS (&Distro), Build Version & Archtitecture.

If you have a distro for which is does not run and you want to contribute changes for it to work, you can Fork the installer repo and create your pull-request when ready to submit your code.

Meanwhile; below the available installer builds, just check the list of verified platforms and a copy of the FSM can be found here.
