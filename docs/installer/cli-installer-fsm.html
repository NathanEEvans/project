CLI Installer - FSM (Frikin' Short Manual)

Grab yourself a copy of the installer here and come back or Fork and build from source. Also come back.

Either and anyway, you will end up with a jar file which you fire up like this 


martijn$ java -jar stormcloud-installer-cli.jar

A startup banner shows up which tells you a little and asks you some

Lets get ready to rumble ...
(type help if you need any)

Stormcloud IDE $

When answering with 'y' the installer will enter into the installer prompt, otherwise it's going to whish you a nice day.

I'm assuming you said 'y', you are now on the installer prompt. 


    
Do you want to run the Stormcloud IDE Installer? [n] y
Stormcloud IDE $

Type help at any time to know what options you have. 


Stormcloud IDE $ help

Use one of these commands: 

help      : Print this list of commands
sanity    : Check if the machine is installation sane
user      : Install the User Home components
server    : Install the Server components
all       : I will do it all
bye       : I will leave you be

Stormcloud IDE $

When running sanity it should output something like below (with your own machine/jdk specs of course).
