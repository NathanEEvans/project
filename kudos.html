Credits's & Kudos

There are many great projects out there which make things like we do on a daily basis possible. Here we would like to says thanks and point you to them.

Dojo Toolkit

Because a poweful and feature rich Javascript Toolkit like Dojo is just around for us to use we have achieved great results on the GUI. No idea how that would ever have happened without Dojo. Pay them a visit and check out the incredible cool things they do.

Apache Tomcat

Since many years very hapily using the Apache Tomcat Servlet Container. This is after the servlet technology itself the most useful piece of software in millennia. That is together with all the other great software seeing daylight at The Apache Software Foundation

Netbeans IDE

We tend to forget and take for granted that we can get a development environment for ziltsj, nada, um zonst, to help us with the coding. Well, we should not not. It's quite remarkable that there are people out there who, simply because they like to create stuff and are not afraid of sharing it, bring you free software. No strings attached. Awesome that is i say.

Spring

If you prefer to not use the automatic generator, push a branch named gh-pages to your repository to create a page manually. In addition to supporting regular HTML content, GitHub Pages support Jekyll, a simple, blog aware static site generator written by our own Tom Preston-Werner. Jekyll makes it easy to create site-wide headers and footers without having to copy them across every page. It also offers intelligent blog support and other advanced templating features.
